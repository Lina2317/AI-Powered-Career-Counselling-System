# AI-Powered Career Counselling System using Data Analytics

## Project Overview

This project is designed to help students identify suitable career paths based on their academic performance, interests, aptitude scores, communication skills, and stream. The system analyzes student data and provides career recommendations through an interactive Power BI dashboard.

## Objectives

* Analyze student data using data analytics techniques.
* Identify suitable career paths based on student profiles.
* Visualize insights through an interactive dashboard.
* Support data-driven career guidance and decision-making.

## Tools & Technologies

* Power BI
* Microsoft Excel
* MySQL
* Python
* Pandas
* NumPy

## Dataset Features

The dataset contains:

* Student_ID
* Gender
* Stream
* Percentage
* Interest
* Aptitude Score
* Communication Score
* Recommended Career

## Dashboard Features

* Total Students KPI
* Average Percentage KPI
* Total Recommended Careers KPI
* Top Recommended Career KPI
* Gender Distribution Analysis
* Stream-wise Student Analysis
* Career-wise Student Analysis
* Interest Distribution Analysis
* Interactive Filters and Slicers

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Analysis
4. Data Visualization
5. Dashboard Development
6. Career Recommendation Analysis

## Key Insights

* Identified the most recommended career paths.
* Analyzed student interests and academic performance.
* Compared career recommendations across different streams.
* Created an interactive dashboard for easy analysis.

## Benefits

* Helps students make informed career decisions.
* Supports career counselling with data-driven insights.
* Reduces confusion in career selection.
* Provides a clear understanding of student strengths and interests.

## Future Scope

## Conclusion

The AI-Powered Career Counselling System demonstrates how data analytics and visualization can be used to support career guidance. The dashboard provides meaningful insights that help students choose suitable career paths based on their skills, interests, and academic performance.
